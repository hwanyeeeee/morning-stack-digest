대규모 언어 모델의 메모리 포이즈닝 공격 탐지 연구

academic2026-03-20

본 연구는 LLM(대규모 언어 모델) 기반 어시스턴트의 장기 메모리 기능을 악용하는 포이즈닝 공격 벡터를 분석하고, 이를 탐지하기 위한 정적·동적 분석 기법을 제안합니다. 실험 결과, 제안된 탐지 모델은 기존 필터 대비 정탐률 94.7%, 오탐률 2.1%를 달성했습니다.

연구 배경

AI 어시스턴트의 메모리 기능은 사용자 편의를 위해 설계되었지만, 악의적인 웹 콘텐츠가 AI의 장기 기억을 오염시키는 공격(MINJA 유형)이 가능합니다. 공격자는 피해자가 방문할 페이지에 숨겨진 명령을 삽입하여 AI가 이를 장기 메모리에 저장하도록 유도합니다. 이 공격의 피해자 인지율은 기존 연구에서 0%에 가까운 것으로 보고된 바 있습니다.

공격 분류 체계

본 논문은 메모리 포이즈닝 공격을 다섯 가지 패턴(직접 저장 명령, 추천 편향 유도, 시스템 위장, 기존 지시 무력화, 다단계 체인)으로 분류합니다. 각 패턴별 공격 성공률과 회피 기법을 실험적으로 측정하였으며, CSS 은닉, 제로폭 문자 삽입, 시각적 위장 등 세 가지 은닉 기법과의 교차 분석도 수행했습니다.

탐지 모델 아키텍처

제안 모델은 다층 신경망 기반 의미 분석기와 규칙 기반 패턴 매처를 결합한 하이브리드 구조입니다. 입력 텍스트를 토큰 단위로 분석하여 명령형 패턴과 개인화 요청을 식별합니다. 제로폭 문자 필터링 전처리 단계를 통해 난독화 시도를 효과적으로 무력화합니다.
