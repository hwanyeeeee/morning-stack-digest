## 카테고리

- `tech_blog` — 언어/프레임워크 실무 가이드 (TypeScript, Rust, Python, Go)
- `news` — 보안 사고, CVE, 데이터 유출 정리
- `review` — IDE, 노트북, 키보드, 앱 리뷰
- `wiki` — 알고리즘·언어·디자인 패턴 백과 항목
- `academic` — NLP/ML/시스템 논문 요약
- `personal` — 시니어 엔지니어 회고·일상
- `corporate` — 기업 소개·채용 페이지
- `career` — 시니어로의 성장 가이드
- `oss` — 오픈소스 릴리즈 노트
- `conference` — 국내외 컨퍼런스 후기
